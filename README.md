# Life Organizing System (C Project)

This project is a menu-based Life Organizing System written in C.  
It helps users organize and track different parts of daily life using simple structures and file handling.

## Features
- **Pantry Management**
  - Add, list, edit, delete items
  - Low stock + expiry alerts

- **Expense Management**
  - Add, list, edit, delete expenses
  - Budget alerts, big expense alerts

- **Health Tracking**
  - Water, calories, sleep, steps
  - Health alerts

- **Movie List**
  - Add, rate, edit, delete movies
  - Low-rated movie alerts

## Technologies Used
- C Programming
- Structures

## How to Compile
gcc src/*.c -Iinclude -o life

## How to Run
./life

- File Handling (text files)
- Modular Programming (`src/` + `include/`)
